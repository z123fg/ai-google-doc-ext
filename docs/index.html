Privacy Policy for AI Search Engine

Last updated: April 9, 2026

AI Search Engine is a Chrome extension for Google Docs that lets users search their own uploaded notes and insert selected answers back into an active document. This privacy policy explains what data the extension accesses, how it is used, and how it is stored.

What the extension accesses

The extension accesses the following data only to provide its core feature:

  • Content from Google Docs pages that the user is actively editing, so the extension can read the current typing context, show relevant search results, and insert selected answers back into the document.
  • Note files the user chooses to upload into the extension for local indexing and retrieval.
  • Local extension settings, such as enabled documents and search behavior preferences.

How data is used

  • To index the user’s uploaded note content for local semantic search.
  • To match the user’s current writing context against the indexed notes.
  • To insert a user-selected answer back into the active Google Doc.
  • To remember local settings and indexed documents across sessions.

How data is stored

Indexed notes and local settings are stored locally in the extension using Chrome extension storage and local browser storage mechanisms. This data is used only to make the extension function and is not sold or shared for advertising purposes.

Data sharing

The extension does not sell user data. The extension does not transfer user data to third parties for purposes unrelated to the extension’s single purpose. The extension does not use user data to determine creditworthiness or for lending purposes.

Remote services

The extension’s executable code is packaged with the extension. If external model or data resources are requested, they are used only to support the extension’s search functionality and not to execute remote code in the extension.

User control

  • Users choose which files to upload for indexing.
  • Users can remove indexed documents from the extension.
  • Users can disable documents from search inside the extension UI.
